O que são pseudoelementos?

Um pseudoelemento CSS é usado para estilizar partes especificadas de um elemento. Por exemplo, ele pode ser usado para estilizar a primeira letra ou linha de um elemento ou inserir conteúdo antes ou depois do conteúdo desse elemento.

Todos os pseudos elementos:

Pseudo elemento Descrição
::first-letter Seleciona a primeira letra do elemento especificado
::first-line Seleciona a primeira linha de um texto com múltiplas linhas
::before Insere um conteúdo antes do conteúdo do elemento selecionado
::after Insere um conteúdo após o conteúdo do elemento selecionado
::selection corresponde à parte de um elemento que é selecionada por um usuário.

Observação: Diferentemente dos pseudo-elementos, pseudo-classes podem ser utilizadas para estilizar um elemento baseado em seu estado.

GIT

Git pronuncia-se é um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo.

O Git foi inicialmente projetado e desenvolvido por Linus Torvalds para o desenvolvimento do kernel Linux, mas foi adotado por muitos outros projetos.

Pseudo elemento ::selection

Selecione parte do texto abaixo, ou pressione ctr+a para selecionar todo o texto

O ::selectionpseudoelemento corresponde à parte de um elemento que é selecionada por um usuário. As seguintes propriedades CSS pode ser aplicada a ::selection: color, background, cursor, e outline.
