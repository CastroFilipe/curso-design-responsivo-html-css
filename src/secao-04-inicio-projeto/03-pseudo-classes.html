Pseudo classes

Uma pseudo-classe CSS é uma palavra-chave adicionada a seletores que especifica um estado especial do elemento selecionado. Por exemplo, :hover pode ser usado para alterar a cor de um botão quando o usuário passar o cursor sobre ele.

Alguns exemplos de uso:

  • Estilize um elemento quando um usuário passa o mouse sobre ele
  • Estilo de links visitados e não visitados de maneira diferente
  • Estilize um elemento quando ele receber foco

Observação: Diferentemente dos pseudo-elementos, pseudo-classes podem ser utilizadas para estilizar um elemento baseado em seu estado.

Observação: Diferentemente das pseudo-classes, pseudo-elementos podem ser usados para estilizar uma parte específica de um elemento.

:link | :visited | :active | :hover

Podem ser utilizados em conjunto para exibir os links de diferentes maneiras. Um link visitado, não visitado, ativo ou no estado hover (quando você passa o mouse sobre ele) pode assumir aspectos (cores, fundos, etc...) diferentes através de CSS.

pseudo-classe Descrição
:link define o estilo do link no estado inicial
:visited estilização de um link que já foi visitado
:active estilização de um link no momento do click
:hover estilização do link ao passar o mouse em cima
